# GB/T 9704-2012 Formatting Rules (格式规范)

## Overview

GB/T 9704-2012 "党政机关公文格式" specifies the format requirements for official documents of party and government organs.

## Page Setup (页面设置)

### Paper Size

```
规格：A4
尺寸：210mm × 297mm
方向：纵向（一般情况）
```

### Margins

```
上白边：37mm ± 1mm
下白边：35mm ± 1mm
左白边：28mm ± 1mm
右白边：26mm ± 1mm

版心尺寸：156mm × 225mm
```

### Line Spacing

```
行距：固定值 28磅（约0.99cm）
每页：22行
每行：28字
```

## Document Elements (公文要素)

### 版头 (Header)

#### 1. 份号

```
位置：版心左上角第一行
字体：黑体，3号
格式：如 "000001" 或 "秘密000001"
说明：涉密公文应标注份号
```

#### 2. 密级和保密期限

```
位置：版心左上角第二行
字体：黑体，3号
格式：如 "机密★5年" 或 "绝密"
说明：涉密公文必须标注
```

#### 3. 紧急程度

```
位置：版心左上角
字体：黑体，3号
格式：特急 或 加急
说明：紧急公文应标注
```

#### 4. 发文机关标志

```
位置：居中
字体：小标宋体，红色
字号：酌定，以醒目、美观、庄重为原则，字号不大于上级机关
格式：
  - 全称：如 "XX省人民政府文件"
  - 简称：如 "XX省府文件"
说明：联合行文时，各机关标志并列排列
```

#### 5. 发文字号

```
位置：发文机关标志下空二行，居中
字体：仿宋_GB2312，3号
格式：机关代字 + 六角括号〔年份〕+ 序号 + 号
示例：国发〔2024〕1号
说明：年份、序号用阿拉伯数字

正确格式：
- XX发〔2024〕1号 ✓
- XX发[2024]1号 ✗
- XX发（2024）1号 ✗
```

#### 6. 签发人

```
位置：居右空一字，编排在发文机关标志下空二行位置
字体："签发人"三字用仿宋_GB2312，3号；签发人姓名用楷体_GB2312，3号
格式：签发人：姓名
说明：
  - 上行文必须标注签发人姓名
  - 联合行文时，一般每行排两个姓名，回行时与上一行第一个签发人姓名对齐
  - 发文字号居左空一字编排，与最后一个签发人姓名处在同一行
```

#### 7. 红色分隔线

```
位置：发文字号之下4mm处
宽度：与版心等宽（156mm）
颜色：红色
说明：
  - 红色分隔线是版头与主体的分界线
  - 线以上为版头，线以下（不含）为主体
  - 一般用红色直线，居中
```

### 主体 (Body)

#### 1. 标题

```
位置：红色分隔线下空二行，居中
字体：方正小标宋简体，2号
格式：发文机关 + 事由 + 文种
示例：国务院关于XX的通知

要求：
  - 一般不加标点符号
  - 一般不超过50字
  - 可分多行排列，回行时词意完整
  - 排列使用梯形或菱形，整齐美观
```

#### 2. 主送机关

```
位置：标题下空一行，顶格
字体：仿宋_GB2312，3号
格式：机关全称、规范化简称或同类型机关统称
示例：各省、自治区、直辖市人民政府

说明：
  - 顶格排列，回行时仍顶格
  - 多个主送机关用顿号或逗号分隔
  - 最后一个主送机关名称后标全角冒号
  - 某些公文可不标注主送机关
```

#### 3. 正文

```
位置：主送机关名称下一行，首行缩进2字符
行距：固定值28磅
每页：22行
每行：28字

字体层级：
  - 正文：仿宋_GB2312，3号
  - 第一层（一、）：黑体，3号（后接顿号）
  - 第二层（（一））：楷体_GB2312（加粗），3号（后不加标点）
  - 第三层（1.）：仿宋_GB2312（加粗），3号（后接半角实心点）
  - 第四层（（1））：仿宋_GB2312（加粗），3号（后不加标点）

层次序号：
第一层：一、二、三、……
第二层：（一）（二）（三）……
第三层：1. 2. 3. ……
第四层：（1）（2）（3）……
```

#### 4. 附件说明

```
位置：正文下空一行，左空二字
字体：仿宋_GB2312，3号
格式：
  附件：1. XXXXX
       2. XXXXX

说明：
  - "附件"二字后标全角冒号和附件名称
  - 附件较多时应标注顺序号
  - 附件名称不加书名号
  - 附件名称后不加标点符号
  - 回行时与上一行附件名称的首字对齐
```

#### 4a. "附件"二字及附件顺序号

```
位置：版心左上角第一行，顶格
字体：黑体，3号
格式：附件1、附件2……
说明：附件与正文一起装订时，在附件页标注此标识
```

#### 4b. 附件标题

```
位置：版心第三行，居中
字体：方正小标宋简体，2号
格式：同公文标题格式
说明：附件顺序号和标题应与附件说明表述一致
```

#### 5. 发文机关署名

```
位置：与正文或附件说明空一行，居右排布
字体：仿宋_GB2312，3号
格式：机关全称或规范化简称
说明：署名应与发文机关标志一致
```

#### 6. 成文日期

```
位置：署名下方
字体：仿宋_GB2312，3号
格式：XXXX年X月X日（阿拉伯数字标注，年月日齐全）
说明：
  - 用阿拉伯数字将年、月、日标全
  - 年份标全称，月、日不编虚位（1不编为01）
  - 不加"年""月""日"之外的标点
  - 右空四字编排
```

#### 6a. 附注

```
位置：成文日期下一行，居左空二字
字体：仿宋_GB2312，3号
格式：加圆括号标注
说明：用于需要说明的其他事项
```

#### 7. 印章

```
位置：骑年盖月（印章覆盖发文机关署名和成文日期）
要求：
  - 印章用红色，不得出现空白印章
  - 印章端正、居中下压发文机关署名和成文日期
  - 署名和日期居印章中心偏下位置
  - 印章顶端上距正文（或附件说明）一行之内
  - 联合行文时，印章间互不相交或相切，每排印章两端不超出版心
```

### 版记 (Footer)

#### 版记分隔线

```
首条分隔线：粗线（推荐高度0.35mm），与版心等宽，位于版记第一个要素之上
中间分隔线：细线（推荐高度0.25mm），与版心等宽
末条分隔线：粗线（推荐高度0.35mm），与版心等宽，与公文最后一面版心下边缘重合
说明：版记中各要素之间用分隔线隔开
```

#### 1. 抄送机关

```
位置：印发机关和印发日期之上一行，左右各空一字
字体：仿宋_GB2312，4号
格式：抄送：XX、XX
说明：最后一个抄送机关名称后标句号
```

#### 2. 印发机关和印发日期

```
位置：末条分隔线之上，印发机关左空一字，印发日期右空一字
字体：仿宋_GB2312，4号
格式：XX印发    XXXX年X月X日印发
说明：印发日期用阿拉伯数字标注，年月日齐全
```

#### 3. 页码

```
位置：版心下边缘之下，一字线上距版心下边缘7mm
字体：4号半角宋体阿拉伯数字
格式：— 1 —（数字左右各放一条一字线）
对齐：单页码居右空一字，双页码居左空一字
说明：版记页前有空白页的，空白页和版记页均不编排页码
```

## Special Formats (特殊格式)

### 信函格式

```
特点：
- 发文机关名称居中
- 发文字号右上角
- 无版头
- 无抄送机关
- 页码在正文下方

适用：
- 函
- 部分批复
```

### 命令格式

```
特点：
- 发文机关标志用全称或规范化简称
- 无主送机关
- 签发人职务和姓名
- 标题可省略

适用：
- 命令
- 令
```

### 纪要格式

```
特点：
- 无主送机关
- 标注出席人员、列席人员
- 正文记录会议情况和议定事项
- 无发文机关署名和印章

适用：
- 会议纪要
```

## Common Errors

### 格式错误

| 错误 | 正确 |
|-----|------|
| 国发[2024]1号（方括号） | 国发〔2024〕1号（六角括号） |
| 二〇二四年三月十五日（中文数字） | 2024年3月15日（阿拉伯数字） |
| I、II、III（罗马数字） | 一、二、三（中文数字） |
| 1、2、3（顿号） | 1. 2. 3.（半角圆点） |
| (一)（半角括号） | （一）（全角括号） |
| 2024.03.15（圆点分隔） | 2024年3月15日 |
| 仿宋/楷体（未指定GB2312） | 仿宋_GB2312/楷体_GB2312 |

### 排版错误

- 行距不一致
- 字体字号不规范
- 标题不居中
- 日期格式错误
- 页码格式错误

## Checklist

```
□ 发文机关标志正确
□ 发文字号格式正确
□ 标题完整、居中
□ 主送机关完整
□ 正文层次序号正确
□ 字体字号规范
□ 发文机关署名正确
□ 成文日期格式正确
□ 页码格式正确
□ 整体排版美观
```
