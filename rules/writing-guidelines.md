# Writing Guidelines (写作规范)

## Language Principles (语言原则)

### 1. 准确 (Accuracy)

```
要求：
- 用词准确，含义明确
- 数据精确，来源可靠
- 概念清晰，界定明确

示例：
✓ 本市共有常住人口2154万人
✗ 本市人口很多
```

### 2. 简洁 (Conciseness)

```
要求：
- 语言精练，不冗长
- 开门见山，直截了当
- 删除冗余词语

示例：
✓ 现将有关事项通知如下
✗ 现在我把有关的事项向你们通知如下
```

### 3. 规范 (Standardization)

```
要求：
- 使用规范术语
- 遵循固定表述
- 符合行文习惯

示例：
✓ 特此通知
✗ 特此给你们通知一下
```

### 4. 得体 (Appropriateness)

```
要求：
- 根据行文关系选择语气
- 上行文谦恭
- 下行文庄重
- 平行文友好
```

## Writing Structure (写作结构)

### 标题撰写

```
基本格式：发文机关 + 事由 + 文种

示例：
国务院关于印发《XX条例》的通知
XX省人民政府关于XX工作的报告
XX部关于XX问题的请示

要求：
1. 准确概括主要内容
2. 使用规范的介词结构
3. 文种准确
4. 一般不加标点
```

### 正文结构

```
基本结构：
[开头] 缘由/背景/依据
[主体] 具体内容
[结尾] 结语/要求

开头方式：
1. 依据式：根据《XX条例》规定，……
2. 目的式：为加强XX工作，……
3. 原因式：鉴于XX情况，……
4. 直接式：现将XX事项通知如下：

结尾方式：
1. 总结式：以上报告，请审阅。
2. 要求式：请遵照执行。
3. 请示式：妥否，请批示。
4. 函复式：特此函复，请知悉。
```

### 层次安排

```
常用结构：

一、总体要求/指导思想
（一）基本原则
（二）主要目标
二、主要任务
（一）任务一
    1. 具体内容
    2. 具体内容
（二）任务二
三、保障措施
（一）组织保障
（二）制度保障
```

## Common Phrases (常用词语)

### 开头用语

```
根据……规定
为……
鉴于……
经研究决定
现就……事项通知如下
```

### 过渡用语

```
一、关于……问题
二、关于……工作
综上所述
现汇报如下
具体如下
```

### 结尾用语

```
通知类：
- 特此通知
- 请遵照执行
- 请认真贯彻执行

报告类：
- 特此报告
- 以上报告，请审阅
- 请予审核

请示类：
- 妥否，请批示
- 当否，请批示
- 以上请示，请予批复

函类：
- 特此函告
- 特此函复
- 请予函复
- 请予支持为盼
```

### 表态用语

```
同意……
批准……
原则同意……
暂不批准……
不予批准……
```

## Document-specific Guidelines (文种写作要点)

### 通知写作要点

```
1. 标题明确具体
2. 受文对象明确
3. 事由清楚
4. 要求具体
5. 结尾规范

正文结构：
[缘由] 为什么要发这个通知
[事项] 通知的具体内容
[要求] 如何执行
```

### 报告写作要点

```
1. 内容真实准确
2. 重点突出
3. 条理清晰
4. 文字简洁

正文结构：
[开头] 报告的缘由
[主体] 报告的内容
  - 基本情况
  - 主要做法
  - 取得成效
  - 存在问题
  - 下步打算
[结尾] 特此报告
```

### 请示写作要点

```
1. 一文一事
2. 事由充分
3. 要求明确
4. 主送机关唯一

正文结构：
[缘由] 为什么要请示
[事项] 请示的具体事项
[意见] 本单位的意见建议
[结尾] 妥否，请批示
```

### 批复写作要点

```
1. 针对性强
2. 态度明确
3. 依据充分
4. 用语规范

正文结构：
[依据] 引用来文
[意见] 批复意见
[结尾] 此复
```

### 函写作要点

```
1. 开门见山
2. 内容具体
3. 语言得体
4. 篇幅简短

正文结构：
[缘由] 发函原因
[事项] 具体内容
[结语] 函告/函复
```

## Punctuation (标点符号)

### 常用规则

```
1. 标题末尾不加标点
2. 层次序号后标点：
   - 第一层：一、（顿号）
   - 第二层：（一）（不加标点）
   - 第三层：1.（半角实心点）
   - 第四层：（1）（不加标点）
3. 主送机关后用冒号
4. 成文日期不加标点
```

## Common Mistakes (常见错误)

### 用语错误

| 错误 | 正确 |
|-----|------|
| 搞好 | 做好 |
| 搞 | 开展/进行 |
| 大约左右 | 大约/约 |
| 截止目前 | 截至目前 |
| 制定规划 | 制订规划 |

### 结构错误

- 请示与报告混淆
- 一文多事
- 层次不清
- 首尾不呼应

### 格式错误

- 标题不完整
- 主送机关遗漏
- 日期格式错误
- 落款不完整

## Quality Checklist

```
□ 标题完整准确
□ 主送机关明确
□ 内容真实准确
□ 层次结构清晰
□ 语言规范简洁
□ 标点符号正确
□ 格式符合规范
□ 落款完整
□ 日期正确
□ 无错别字
```
